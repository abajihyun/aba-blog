# Hexo 사용법
bash 커맨드로 되어있는 부분은, 터미널 창, aba-blog 디렉토리에서 실행한다.

1. local 서버 실행
aba-blog 디렉토리 안에서 다음의 명령을 실행한다.
```bash
$ hexo server
```

2. 새로운 포스트 생성
다음 명령어로 새로운 포스트를 만들고, 마크다운 문법으로 내용을 작성한다.
```bash
$ hexo new {page 제목}
```

3. 포스트로 정적 리소스 생성

4. 블로그 운영 레파지토리 배포
